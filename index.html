Neo.github.io

Number Extraction in the Open Domain

Download .zip Download .tar.gz View on GitHub

The goal of this project is to create knowledge bases with numerical attributes of entities by automatically extracting such data from unstructured web documents. The extractors will be bootstrapped from existing structured repositories of numerical data such as Wikipedia infoboxes, DBPedia or Freebase. Extractors could either be specialized for a known attribute of an entity type “e.g. Population of a city” or discover new attributes “e.g. number of annual migrants to a city”.
